# Game-Theory

II dalis (`E_Vaitkeviciute_ND2` ir `E_Vaitkeviciute_ND2.html`). Apima temas: dalinės informacijos lošimai, specialūs nekooperatinių lošimų skyriai, koalicinės formos lošimai.
